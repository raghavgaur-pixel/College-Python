# Regex
It is a powerful tool for various kinds of string manipulation, these are the text string used for describing a search pattern to extract information from text such as code, file, log, spreadsheet or even documents.
Regex are domain specific language present as a library in most of modern programming language, besides python.
A regex is a special sequence of characters that help to match or find string in another string.
In python, regex can be accessed using "re" module, which comes as a part of standard library.

										for eg	:	[^0-3]	=	means any number except 0, 1, 2 or 3

# Special Sequences
Special Sequence is 	: \\ followed by one of the characters, which has a special meaning as given below
				1. \\A	-  Returns a match, if the specified characters are at the beginning of the string.
				2. \\b	-  Returns a match, where the specified characters are at the beginning or at the end of a word. {small r in the beginning makes sure that string is treated as raw string}
				3. \\B	-  Returns a match, where the specified characters are present, but not at the beginning of the word.
				4. \\d	-  Returns a match, where strings contains digits (0-9).
				5. \\D	-  Returns a match, where string does not contain digits.
				6. \\s		-  Returns a match, where string contains a white space character.
				7. \\S	-  Returns a match, where string does not contain a white space character.
				8. \\w	-  Returns a match, where string contains any word character (a-z, 0-9, A-Z, _).
				9. \\W	-  Returns a match, where the string does not contain any word character.
				10. \\Z	-  Returns a match, if the specified characters are at the end of the string.

# Data Encapsulation :
- Wrapping up of data and functions into a single unit (called class) is known as encapsulation.
- Data is not accessible to the outside world, only the functions which are wrapped inside the class can access it.
- These functions provide interface b/w object's data and the program. This insulation of data from Direct Access by the program is called data hiding or information hiding.

# Data Abstraction :
- Refers to the act of representing essential features without including the background details or explanation.
- Classes use this concept and are defined as a list of abstract attributes, such as size, weight and cost and functions operate on these attributes.
- The attributes, are sometimes called data members and functions that operate on these data are known as methods or member functions.
- Classes are also known as Abstract Data Types (ADT).

# Inheritance :
- It is the process by which object of 1 class require the properties of objects of another class.
- The concept of inheritance provides the idea of reusability, this means additional features can be added to an existing class without modifying it.
- This is possible by deriving a new class from the existing one, the new class will have combined features of both the classes.

# Polymorphism :
- It allows object having different internal structures share the same external interface.
- This means that a general class of operations may be accessed in the same manner even though specific actions associated with each operation may differ.
- Polymorphism is extensively used in implementing inheritance.
